Element ADEOfBridgePart

ADEOfBridgeRoom acts as a hook to define properties within an ADE that are to be added to a BridgeRoom.

Element AbstractBridge

AbstractBridge is an abstract superclass representing the common attributes and associations of the classes Bridge and BridgePart.

Element class

Indicates the specific type of the Bridge or BridgePart.

Element function

Specifies the intended purposes of the Bridge or BridgePart.

Element usage

Specifies the actual uses of the Bridge or BridgePart.

Element isMovable

Indicates whether the Bridge or BridgePart can be moved to allow for watercraft to pass.

Element bridgeConstructiveElement

Relates the constructive elements to the Bridge or BridgePart.

Element bridgeInstallation

Relates the installation objects to the Bridge or BridgePart.

Element bridgeRoom

Relates the rooms to the Bridge or BridgePart.

Element bridgeFurniture

Relates the furniture objects to the Bridge or BridgePart.

Element address

Relates the addresses to the Bridge or BridgePart.

Element adeOfAbstractBridge

Augments AbstractBridge with properties defined in an ADE.

Element Bridge

A Bridge represents a structure that affords the passage of pedestrians, animals, vehicles, and service(s) above obstacles or between two points at a height above ground. [cf. ISO 6707-1]

Element bridgePart

Relates the bridge parts to the Bridge.

Element adeOfBridge

Augments the Bridge with properties defined in an ADE.

Element BridgeConstructiveElement

A BridgeConstructiveElement is an element of a bridge which is essential from a structural point of view. Examples are pylons, anchorages, slabs, beams.

Element class

Indicates the specific type of the BridgeConstructiveElement.

Element function

Specifies the intended purposes of the BridgeConstructiveElement.

Element usage

Specifies the actual uses of the BridgeConstructiveElement.

Element adeOfBridgeConstructiveElement

Augments the BridgeConstructiveElement with properties defined in an ADE.

Element BridgeFurniture

A BridgeFurniture is an equipment for occupant use, usually not fixed to the bridge. [cf. ISO 6707-1]

Element class

Indicates the specific type of the BridgeFurniture.

Element function

Specifies the intended purposes of the BridgeFurniture.

Element usage

Specifies the actual uses of the BridgeFurniture.

Element adeOfBridgeFurniture

Augments the BridgeFurniture with properties defined in an ADE.

Element BridgeInstallation

A BridgeInstallation is a permanent part of a Bridge (inside and/or outside) which does not have the significance of a BridgePart. In contrast to BridgeConstructiveElements, a BridgeInstallation is not essential from a structural point of view. Examples are stairs, antennas or railways.

Element class

Indicates the specific type of the BridgeInstallation.

Element function

Specifies the intended purposes of the BridgeInstallation.

Element usage

Specifies the actual uses of the BridgeInstallation.

Element adeOfBridgeInstallation

Augments the BridgeInstallation with properties defined in an ADE.

Element BridgePart

A BridgePart is a physical or functional subdivision of a Bridge. It would be considered a Bridge, if it were not part of a collection of other BridgeParts.

Element adeOfBridgePart

Augments the BridgePart with properties defined in an ADE.

Element BridgeRoom

A BridgeRoom is a space within a Bridge or BridgePart intended for human occupancy (e.g. a place of work or recreation) and/or containment (storage) of animals or things. A BridgeRoom is bounded physically and/or virtually (e.g. by ClosureSurfaces or GenericSurfaces).

Element class

Indicates the specific type of the BridgeRoom.

Element function

Specifies the intended purposes of the BridgeRoom.

Element usage

Specifies the actual uses of the BridgeRoom.

Element bridgeFurniture

Relates the furniture objects to the BridgeRoom.

Element bridgeInstallation

Relates to the installation objects to the BridgeRoom.

Element adeOfBridgeRoom

Augments the BridgeRoom with properties defined in an ADE.
