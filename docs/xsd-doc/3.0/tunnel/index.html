Element AbstractTunnel

AbstractTunnel is an abstract superclass representing the common attributes and associations of the classes Tunnel and TunnelPart.

Element class

Indicates the specific type of the Tunnel or TunnelPart.

Element function

Specifies the intended purposes of the Tunnel or TunnelPart.

Element usage

Specifies the actual uses of the Tunnel or TunnelPart.

Element tunnelConstructiveElement

Relates the constructive elements to the Tunnel or TunnelPart.

Element tunnelInstallation

Relates the installation objects to the Tunnel or TunnelPart.

Element hollowSpace

Relates the hollow spaces to the Tunnel or TunnelPart.

Element tunnelFurniture

Relates the furniture objects to the Tunnel or TunnelPart.

Element adeOfAbstractTunnel

Augments AbstractTunnel with properties defined in an ADE.

Element HollowSpace

A HollowSpace is a space within a Tunnel or TunnelPart intended for certain functions (e.g. transport or passage ways, service rooms, emergency shelters). A HollowSpace is bounded physically and/or virtually (e.g. by ClosureSurfaces or GenericSurfaces).

Element class

Indicates the specific type of the HollowSpace.

Element function

Specifies the intended purposes of the HollowSpace.

Element usage

Specifies the actual uses of the HollowSpace.

Element tunnelFurniture

Relates the furniture objects to the HollowSpace.

Element tunnelInstallation

Relates the installation objects to the HollowSpace.

Element adeOfHollowSpace

Augments the HollowSpace with properties defined in an ADE.

Element Tunnel

A Tunnel represents a horizontal or sloping enclosed passage way of a certain length, mainly underground or underwater. [cf. ISO 6707-1]

Element tunnelPart

Relates the tunnel parts to the Tunnel.

Element adeOfTunnel

Augments the Tunnel with properties defined in an ADE.

Element TunnelConstructiveElement

A TunnelConstructiveElement is an element of a Tunnel which is essential from a structural point of view. Examples are walls, slabs, beams.

Element class

Indicates the specific type of the TunnelConstructiveElement.

Element function

Specifies the intended purposes of the TunnelConstructiveElement.

Element usage

Specifies the actual uses of the TunnelConstructiveElement.

Element adeOfTunnelConstructiveElement

Augments the TunnelConstructiveElement with properties defined in an ADE.

Element TunnelFurniture

A TunnelFurniture is an equipment for occupant use, usually not fixed to the tunnel. [cf. ISO 6707-1]

Element class

Indicates the specific type of the TunnelFurniture.

Element function

Specifies the intended purposes of the TunnelFurniture.

Element usage

Specifies the actual uses of the TunnelFurniture.

Element adeOfTunnelFurniture

Augments the TunnelFurniture with properties defined in an ADE.

Element TunnelInstallation

A TunnelInstallation is a permanent part of a Tunnel (inside and/or outside) which does not have the significance of a TunnelPart. In contrast to TunnelConstructiveElement, a TunnelInstallation is not essential from a structural point of view. Examples are stairs, antennas or railings.

Element class

Indicates the specific type of the TunnelInstallation.

Element function

Specifies the intended purposes of the TunnelInstallation.

Element usage

Specifies the actual uses of the TunnelInstallation.

Element adeOfTunnelInstallation

Augments the TunnelInstallation with properties defined in an ADE.

Element TunnelPart

A TunnelPart is a physical or functional subdivision of a Tunnel. It would be considered a Tunnel, if it were not part of a collection of other TunnelParts.

Element adeOfTunnelPart

Augments the TunnelPart with properties defined in an ADE.
