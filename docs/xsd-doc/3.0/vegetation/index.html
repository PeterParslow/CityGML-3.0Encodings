Element ADEOfAbstractVegetationObject

ADEOfAbstractVegetationObject acts as a hook to define properties within an ADE that are to be added to AbstractVegetationObject.

Element ADEOfPlantCover

ADEOfPlantCover acts as a hook to define properties within an ADE that are to be added to a PlantCover.

Element ADEOfSolitaryVegetationObject

ADEOfSolitaryVegetationObject acts as a hook to define properties within an ADE that are to be added to a SolitaryVegetationObject.

Element AbstractVegetationObject

AbstractVegetationObject is the abstract superclass for all kinds of vegetation objects.

Element adeOfAbstractVegetationObject

Augments AbstractVegetationObject with properties defined in an ADE.

Element PlantCover

A PlantCover represents a space covered by vegetation.

Element class

Indicates the specific type of the PlantCover.

Element function

Specifies the intended purposes of the PlantCover.

Element usage

Specifies the actual uses of the PlantCover.

Element averageHeight

Specifies the average height of the PlantCover.

Element minHeight

Specifies the minimum height of the PlantCover.

Element maxHeight

Specifies the maximum height of the PlantCover.

Element adeOfPlantCover

Augments the PlantCover with properties defined in an ADE.

Element SolitaryVegetationObject

A SolitaryVegetationObject represents individual vegetation objects, e.g. trees or bushes.

Element class

Indicates the specific type of the SolitaryVegetationObject.

Element function

Specifies the intended purposes of the SolitaryVegetationObject.

Element usage

Specifies the actual uses of the SolitaryVegetationObject.

Element species

Indicates the botanical name of the SolitaryVegetationObject.

Element height

Distance between the highest point of the vegetation object and the lowest point of the terrain at the bottom of the object.

Element trunkDiameter

Specifies the diameter of the SolitaryCityObject's trunk.

Element crownDiameter

Specifies the diameter of the SolitaryCityObject's crown.

Element rootBallDiameter

Specifies the diameter of the SolitaryCityObject's root ball.

Element maxRootBallDepth

Specifies the vertical distance between the lowest point of the SolitaryVegetationObject's root ball and the terrain surface.

Element adeOfSolitaryVegetationObject

Augments the SolitaryVegetationObject with properties defined in an ADE.

The Vegetation module supports representation of vegetation objects with vegetation-specific thematic classes. CityGML’s vegetation model distinguishes between solitary vegetation objects like trees, and vegetation areas which represent biotopes like forests or other plant communities.
